# GitHub Markdown Memo 📝
GitHubで使用できるマークダウンやフォーマットの構文を自分なりにまとめとく備忘録。<br>
<br>

## 見出し - Heading -
  ```
  # h1
  ## h2
  ### h3
  ~
  ###### h6
  ```
  # h1
  ## h2
  ### h3
  ~
  ###### h6
<br>

## テキスト書式　- Text Style -
|スタイル|構文|出力|
:-:|:-:|:-:
|太字　Bold|`** **` or `__ __`||
|斜体　Italic|`* *` or `_ _`||
|取り消し線|`~~ ~~`||
|太字内の斜体|||
|太字かつ斜体|||
